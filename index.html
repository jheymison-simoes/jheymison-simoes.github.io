Mottu Locações de Veículo

Inicio: 12/04/2021 á atualmente

Descrição: Atuação no desenvolvimento e melhorias de software utilizando stack .Net e Blazor (SPA). Principais projetos: - Criação de um totem atendimento para atendimento ao cliente, onde o cliente insere o CPF, autêntica e é redirecionado a tela de serviços disponíveis. Também foi criado a parte para que o atendente visualizasse os atendimentos por fila e pudesse atendê-lo de acordo com a opção de serviço escolhida pelo cliente. Todo o back end foi criado com .Net e o frontend foi feito utilizando o Blazor Server (SPA). - Participação na criação de micro serviço para agendamentos. Foi criado um sistema completo para que o cliente pudesse realizar um agendamento para retirar a moto alugada ou realizar manutenção. No sistema é possível cadastrar um evento, definir um local para este evento, cadastrar expedientes informando o horario de funcionamento do agendamento e pra cada horário de funcionamento é possível definir quantos agendamentos por hora será possível, limite de dias que será apresentado ao cliente, intervalo de um agendamento para o outro, período de espera e etc. O sistema disponibiliza ao cliente somente os dias disponíveis de acordo com as configurações definidas, ou seja, não irá mostrar dias em que a quantidade máxima de agendamento já foi ultrapassada. O sistema de acorodo com o dia escolhido, disponibiliza apenas os horários em que a quantidade maxima de agendamento para a hora escolhida nao foi ultrapassada. Todo o sistema é configurável e escalável. Nesse projeto utilizamos .Net, Docker, Design Patter Command, RabbitMQ, Hangfire, Refit, PostegreSQL, MongoDB e Entity Framework. - Outros: - Realização de melhorias no processo de operação, como: levar uma moto para a manutenção, realizar troca de moto, entrega de moto, recolher moto.
